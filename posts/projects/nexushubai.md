**NexusHub AI** is a Next.js project built as an AI hub — a single
dashboard for aggregating AI tools and services rather than juggling them
across separate sites.

## Stack

- Next.js (App Router) + TypeScript
- React
- Deployed on Vercel

## Links

- Source: [github.com/g4rrzx/nexushubai](https://github.com/g4rrzx/nexushubai)

## Notes

NexusHub fits the theme running through much of my recent work — making AI
tooling easier to reach. The idea is a central hub where different AI
capabilities live behind one consistent interface, so switching tasks
doesn't mean switching tabs and tools.

It's built on the standard Next.js App Router + TypeScript stack and
deployed on Vercel, keeping the foundation conventional so the focus stays
on the aggregation experience.
