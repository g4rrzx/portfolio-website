**MiMo CodeAssist** (MiMo Prompt Library) is a curated collection of 100+
tested prompts optimized for Xiaomi MiMo AI models. Each prompt is tagged
with the best-fit MiMo model and grouped into Coding, Reasoning, Creative,
and Analysis categories, with one-click copy for instant use. Built for the
**Xiaomi MiMo 100T Token Creator Incentive Program**.

## Stack

- Vanilla HTML / CSS / JavaScript (zero dependencies)
- Xiaomi MiMo series (Reasoner v2.5, Chat v1, Code v1)
- Hermes Agent framework
- Deployed on Vercel

## Links

- Live: [mimo-codeassist.vercel.app](https://mimo-codeassist.vercel.app)
- Source: [github.com/g4rrzx/mimo-codeassist](https://github.com/g4rrzx/mimo-codeassist)

## Notes

The whole thing is intentionally dependency-free — pure HTML/CSS/JS — so it
loads instantly and is trivial to host. Prompts are filterable by category
and each carries a per-prompt model recommendation, so a developer isn't
just copying a prompt but learning *which* MiMo model it's tuned for.

The value is curation: rather than guessing at prompt phrasing, you get
tested, optimized prompts for common tasks, which is exactly what speeds up
day-to-day work with the MiMo models.
