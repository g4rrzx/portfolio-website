**Farcaster Roulette** is an interactive Web3 mini-app featuring daily
spins, automated quest verification, and a premium neon UI with gamified
reward systems on the [Farcaster](https://farcaster.xyz) network.

## Stack

- Next.js
- React
- TypeScript
- TailwindCSS

## Links

- Live: [rouletee.vercel.app](https://rouletee.vercel.app)

## Notes

Quests are verified automatically against Farcaster signals (follows,
recasts, likes) so the loop is "do a thing, spin the wheel, win a reward"
with zero manual moderation. The UI leans into the neon/arcade aesthetic
to reinforce the luck-based feel while the underlying state machine keeps
rewards deterministic.
